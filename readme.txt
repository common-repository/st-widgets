== Description ==

Adding this widget to your website will ease the process of installing ServiceTitan Widgets.
This is developed to help you add the source code to your website quicker and with no issues.

Web Scheduler Widget

Enabling this will add the Web Scheduler source code to your website. In order to have the widget up and working you will still need to add the button code to it.

Chat To Text Widget

Enabling this will add the Chat to Text source code to your website and have the widget up and running.
